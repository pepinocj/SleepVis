mongodatabase

Establish connection to a mongo database.

returns mongo database object.

connect = mongodatabase(server, port, databasename)
%   With this syntax, it is assumed that the MONGO SERVER(s)
%   doesn't expect a username and password to establish a connection
%   And the USER has appropriate permissions to work on provided
%   DATABASENAME

connect = mongodatabase(server, port, databasename, 'Name', 'Value')
%   With this syntax it is assumed that MONGO server(s) expects a username and password
%   and User is assumed to have READ/ WRITE permissions for selected databases only.
  • server: MATLAB CHAR or CELL of STRINGS used to specify one or more server names
  • port: SCALAR or CELL ARRAY of DOUBLE to specify corresponding port number(s)
  • databasename: MATLAB CHAR used to specify the database name
  • PARAM-VALUE pairs can be:
   USERNAME:       USERNAME is a string used for authenticating a user for a given database name.
   PASSWORD:       PASSWORD is a string used for authenticating a user for a given username
   AUTHMECHANISM:  Can be one of the following:
                   - PLAIN:       verifies user credentials using LDAP service
                   - MONGODB_CR:  is a challenge-response mechanism that authenticates users through passwords
                   - SCRAM_SHA_1: verifies supplied user credentials against the user's name, password and database.
                   If not provided, Creates a MongoCredential instance with an unspecified mechanism.
                   The client will negotiate the best mechanism based on the version of the server that the client is authenticating to.

Example

conn = mongodatabase('localhost', 27017, 'testdb')
% Establishes a connection to LOCALHOST and returns a MONGO DATABASE OBJECT
% for database 'testdb'

conn = mongodatabase('localhost', 27017, testdb, 'UserName', 'testdbuser', 'Password', 'pass1')
% Establishes a connection to LOCALHOST,
% authenticates 'testdbuser' against 'testdb'
% Authentication Mechanism is negotiated between MATLAB and Mongo SERVER, and returns

conn = mongodatabase({'localhost', 'remotehost'}, {27017, 27018}, testdb, 'UserName', 'testdbuser', 'Password', 'pass1')
% Establishes a connection to LOCALHOST and REMOTEHOST on PORT 27017 and 27018 respectively,
% authenticates 'testdbuser' against 'testdb'
% Authentication Mechanism is negotiated between MATLAB and Mongo SERVER, and returns

conn = mongodatabase('localhost', 27017, testdb, 'UserName', 'testdbuser', 'Password', 'pass1', 'AuthMechanism', 'MONGODB_CR')
% Establishes a connection to LOCALHOST on PORT 27017,
% authenticates 'testdbuser' against 'testdb'
% Authentication Mechanism used is 'MONGODB_CR'.
